# Frameworks para teste de unidade

- MSTest;
- NUnit;
- XUnit.

## XUnit

- Escrito pelos criados do NUnit;
- Compatível com Visual Studio;
- Utilizado pelo time de desenvolvimento do .NET Core e ASP.NET;
- Instalação: `Install-Package xunit`
- Instalação para reconhecimento do visual studio: `Install-Package xunit.runner.visualstudio`
