# Backend for frontend

Uma API criada especificamente para uma tela frontend. Seu objetivo é retornar  dados para a montagem da tela, isso inclui: condição para renderização de elementos, labels, textos, etc...

Um BFF atende as necessidades de uma interface.

Um exemplo legal é apps mobile. Alterar o frontend de um app resultará em ter que lançar uma nova versão na loja. Esse processo demora. Agora, se essa alteração pode ser feita via backend, então bastará subir uma nova versão do app. (Server-Driven UI)
